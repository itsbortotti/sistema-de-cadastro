# Backend — Governança Financeira de Projetos

API REST em Node.js/Express. Sessão por cookie, documentação Swagger.

## Rodar

```bash
npm install
node server.js
```

- Servidor: http://localhost:3001  
- Swagger: http://localhost:3001/api-docs
